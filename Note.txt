Global Attributes:

	Global attributes can be applioed to any HTML elements and are not restricted to specific tags.

	Examples include "class". "id", "style", "title", and "lang".


Comments:
    HTML comments are used to insert notes to a web page.
    Single line comment:
         
         <!-- This is a single line comment. -->

    Multi-line comment:

        <!-- This is a multi-line comment.
            Multi[le line commengts can include line breaks
            and also extra   spaces..........]
              -->
